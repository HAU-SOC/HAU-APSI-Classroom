# HAU-APSI-Classroom

A Jekyll-based classroom site for HAU APSI students, hosted via GitHub Pages.

## Pages

- **Home** (`/`) — Announcements, quick links, activities table, and contact info.
- **Resources** (`/resources`) — Learning modules, course syllabus, reference materials, submission portal, and FAQ.
